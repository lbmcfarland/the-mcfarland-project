Web Development

Web Development. The hard truth about learning to develop web pages is that you must actually write code to truly learn it. W3Schools.com is a great source for code language and examples. This part of the Study Companion will provide some syntax, basic definitions, and a list of resources to assist your learning. When in doubt, or you want to find something specific, Google "how do I _______ in html (or css/bootstrap/etc.)."

  • Basic Terminology
  • Web Development Resources

Basic Terminology

  • HTML - hypertext markup language
  • CSS - cascading style sheet
  • API - application program interface
  • HTTP - hypertext transfer protocol
  • FTP - file transfer protocol
  • UX - user experience
  • UI - user interface

Web Resources

Text Editors. Applications you use to create the text files which are your html/css/js documents. There are many options, but here are two:

  • Notepad - basic text word processor preinstalled on PCs
  • VS Code - what we used in class, this is a text editor enhanced for web code writing

Websites. Here are links to the websites we used frequently in class.

  • w3schools.com - your one-stop shopping for online code learning
  • Get Bootstrap - the site with the bootstrap head link and many examples
  • Stack Overflow - the online collaboration forum for web developers
  • Udemy - a large online learning site
