Basic Terminology

Application. Simply put, an app is a type of software that allows you to perform specific tasks. Applications for desktop or laptop computers are sometimes called desktop applications, while those for mobile devices are called mobile apps. When you open an application, it runs inside the operating system until you close it. "Application," "app," "computer program," and "software" are expressions that generally mean the same thing.

Hardware. Computer hardware is the collection of physical parts of a computer system. This includes the computer case, monitor, keyboard, and mouse. It also includes all the parts inside the computer case, such as the hard disk drive, motherboard, video card, and many others. Computer hardware is what you can physically touch.

Desktop Elements. The image above annotates some important elements of the desktop that will make using your personal computer (PC) easier. You should become completely familiar with all the aspects of your desktop - this will require taking time and playing around with clicks. If you click on the image, it will open larger in a new window.

Task Bar. Along the bottom of the desktop is the Task Bar. All open applications are shown there as well as any application icons that are pinned to it. If you look closely, you will see that some of the icons are underscored. These are the ones running. I like to pin applications I frequently use to the task bar. They can be opened with one click and are always present without needing to open start or closing windows to get to the desktop icons.

Start Button. If you press the start button, the start menu will expand. Here is where you will find the sleep/shut down/restart feature, the list of all programs, and a panel in which you may pin applications (pin to start) for easy access.

Search Bar. The search bar on the task bar is useful because it not only searches your filenames and folder names, but it also searches inside the files for your search text. This is useful if you can't remember what you named a document, but you do remember some internal text.

Folders

For the web development and Microsoft Azure blocks of the TCP curriculum, you need to be adept at creating and manipulating folders and files on your computer. Organizing your digital files will not only make your work more efficient, but it will be necessary when creating the required file structures for websites and the Azure cloud computing environment.

Click image for larger view in new browser tab.

File Explorer. Alternatively referred to as Windows Explorer or Explorer, File Explorer is a file browser found in every version of Microsoft Windows since Windows 95. It is used to navigate and manage the drives, folders, and files on your computer. The icon for File Explorer is the manila folder, and the icon is usually automatically pinned to the task bar.

Computer Folder. In computers, a folder has the look and feel of a manila file folder and they can be stored in any location of your operating system including external media storage devices. The purpose for these folders is to create a virtual filing system to organize and categorize your files.

Root Folder. Also called the root directory or sometimes just the root, of any partition or folder is the highest directory in the hierarchy. The root directory contains all other folders in the drive or folder, and can, of course, also contain files.

Files

A file is the common storage unit in a computer, and all programs and data are 'written' into a file and 'read from a file. Folders provide a method for organizing files much like a manila file folder contains paper documents in a file cabinet. In fact, files that contain text are often called documents.

Naming Files. The filename is the name you give your computer file along with the file extension that identifies the type of file it is. Sometimes a specific filename is required (such as the index.html file used to identify the home page in the root folder of a web server). When choosing a filename for a file you create, try to make it relevant to your work so you can remember it and organize it effectively.

File Extensions. A file extension (or simply "extension") is the suffix at the end of a filename that indicates what type of file it is. There are many file formats out there, but here is a list of common types you will often encounter.

  • css - Cascading style sheet text file
  • docx - Microsoft Word document
  • gif - Image format: Graphics Interchange Format
  • html - web page source text
  • jpeg - (or jpg) Image format: JPEG
  • pdf - Acrobat - Portable document format
  • png - Image format: Portable Network Graphics
  • pptx - Powerpoint document
  • xlsx - Excel spreadsheet file
