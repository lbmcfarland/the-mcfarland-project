Study Companion Topics

Computer Basics. The Technology Career Program (TCP) is a computer-centric course, so having a solid understanding of how you interface with the computer is essential. This section will go over topics such as creating and storing files and using the desktop features. Feel free to skip this section if your "geek" is sufficiently fluent in this area!

Scrum. The TCP curriculum includes Scrum Master certification training. As it happens, the exam provided by Scrum.org is fairly difficult. This section of the Study Companion will go over the test format and suggest studying and test-taking tips. A list of "must-know" questions and links to study reference materials will also be provided.

Web Development. During this TCP course block, you will learn the basics of developing web pages. This includes using html, css, javascript, and bootstrap. In this section of the Companion, I will cover basic terminology and provide code snippets you can use when developing your own pages. I will also provide links to the reference material given in class and in my own search for answers.

Microsoft Azure. The final TCP block of instruction is on Microsoft Azure. As we go through this course, I will add study questions and reference materials to this section of the site.

Professional Development. We were given many tools to improve professional skills and to enhance our professional brand. This section of the Companion provides a summary of the various topics covered with links to documents and resources to continue this part of our journey.
